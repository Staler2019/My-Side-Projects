# Windows environment installation

1. Install Git: https://git-scm.com/
2. Run the following in your terminal

   ```.bat
   cd C:\
   mkdir Developments

   cd %USERPROFILE%
   mkdir Workspace

   cd Workspace
   mkdir "My files"
   mkdir Projects
   mkdir Tools

   cd Projects
   mkdir Private
   mkdir Public
   mkdir Docker

   git clone https://github.com/Staler2019/Code.git
   ```
